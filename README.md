# Auron Strategic Core

Cortex OS - Auron: Bas Mimar ve Stratejist ajaninin bagimsiz cekirdek projesi.

## Hakkinda

Auron, Cortex OS ekosisteminin stratejik analiz ve mimari planlama motorudur. ReAct (Tool Calling) mimarisi ile derin analiz yeteneklerine sahiptir.

## Yetenekler

- Tree of Thoughts tabanli stratejik analiz
- First Principles & SCAMPER & MECE analiz cerceveleri
- Varsayim Haritalama ve Stres Testi
- Olceklenebilirlik ve Vizyon planlama
- Atlas ajani icin gorev plani uretimi
